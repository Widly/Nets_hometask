Общим элементом для всех заданий является передача данных между двумя приложениями, протокол, платформу взаимодействия и язык реализации можно использовать по выбору.
1. Алгоритм CRC. Реализовать передачу произвольного сообщения с рассчитанной RCR-суммой и внесенной ошибкой. Получатель должен идентифицировать ошибку по CRC.
2. Код Хемминга. Реализовать передачу произвольного сообщения с контрольными битами по алгоритму Х., получатель должен идентифицировать ошибку  и исправить ее.
3. Передача сжатого сообения по алгоритму Шеннона-Фано.
4. Передача сжатого сообения по алгоритму LZ77,LZ78,LZW.
5. Реализовать сжатие алгоритмом JPEG.
